Mr. The Movie Renderer is a React application I developed and designed using the RESTful API, The Movie Database (TMDb).

On load, the landing page performs an API request to TMDb to populate the current popular movies and their titles.

The search bar allows users to search for a specific movie title.

On movie click, a module pops up centered in the viewport. At this time, another API request is made to TMDb to grab the movie poster image, movie title, description, original release date and the voters average.
