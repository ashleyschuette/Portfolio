Bind Effort is a full stack application that helps eliminate hiring bias in the work place by hiding applicant demographics. Created on a 3 person team using:
- Vue.js
- Vuex
- JavaScript
- HTML
- CSS
- Node.js
- PostgreSQL
- Auth0

You can read my contributions below and click the image to see this application in action.

I took initiative to create wireframes using Adobe XD, which was used as a guide for design decisions.

My contributions to this project were:

- Designing the logo in Photoshop
- Implementing a typewriter npm package and writing javascript, so it would fit our needs
- Coding a majority of the CSS and HTML for every component
- Implementing Vue animations in conjunction with CSS animations on the resume selector page (pass or shortlist buttons)
- Writing javascript to allow the resume and job editors to conditionally render
- Setting up routes
- Customizing Material UI components
